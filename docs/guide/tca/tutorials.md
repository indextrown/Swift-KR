# 튜토리얼

원문: [The Composable Architecture](https://swiftpackageindex.com/pointfreeco/swift-composable-architecture/main/documentation/composablearchitecture)

공식 문서는 다음 두 개의 대화형 튜토리얼을 제공합니다.

## Meet the Composable Architecture

[Meet the Composable Architecture](https://swiftpackageindex.com/pointfreeco/swift-composable-architecture/main/tutorials/meetcomposablearchitecture)는 TCA의 상태, 액션, reducer, Store, 테스트, 내비게이션을 단계별로 익히는 입문 튜토리얼입니다.

## Building SyncUps

[Building SyncUps](https://swiftpackageindex.com/pointfreeco/swift-composable-architecture/main/tutorials/buildingsyncups)는 복잡한 앱을 처음부터 만들면서 도메인 모델링, 내비게이션, 의존성, 테스트를 다루는 심화 튜토리얼입니다.
